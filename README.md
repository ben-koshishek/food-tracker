# Food Tracker

Calorie and macro tracker.

**[Live Demo →](https://ben-koshishek.github.io/food-tracker/)**

---

## Features

- **Barcode lookup** — Fetch nutrition data from [Open Food Facts](https://world.openfoodfacts.org/)
- **Personal food database** — Save foods locally for quick access
- **Meal templates** — Reusable meals and full day plans
- **Calorie calculator** — TDEE-based goal estimation with 5 presets (cut, lean recomp, recomp, maintain, bulk)
- **Goal tracking** — Daily targets for calories and macros
- **Quality scores** — Nutri-Score and NOVA levels
- **Price tracking** — Compare store prices per kg
- **Export/Import** — Backup data as JSON
- **Local storage** — Data stays in your browser (no sync between devices yet)

---

## Tech

React, TypeScript, Vite, Tailwind CSS, Dexie.js
